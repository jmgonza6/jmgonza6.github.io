LibLAMMPS version 0.1

This is a library of tools targeted for graphene simulations conducted in the classical Molecular Dynamics package LAMMPS. It is written in a mix of Ansi C and C++ where appropriate for speed, generality or space efficiency. Several methods are provided to generate LAMMPS input files of several geometries, manipulate LAMMPS input/dump files, and process the log files.

It can be built as either an archive i.e., liblammps.a or as a shared object, liblammps.so. See the Makefile in the lib/ directory. Several other libraries are linked to this one such as a memory and error management class. These are stand alone and can be built separately with the Makefile in lib/.

Several examples of how to use the library are contained in the examples/ directory.
