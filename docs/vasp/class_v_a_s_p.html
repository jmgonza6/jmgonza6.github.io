Coordinates are stored in a struct called output
the simulation cell is stored in a struct called outbox, accessible via dump()
.

Parameters
[in]unitcell- Number of atoms in a unit-cell
[in]geometry- Sample geometry
  • membrane or orthogonal or triclinic
[in]periodic- Periodic boundary conditions
  • true or false
[in]nx- Number of unit cells in x-direction
[in]ny- Number of unit cells in y-direction
[in]nz- Number of unit cells in z-direction
[in]rcc- Equilibrium graphene bond length, in Å
[in]deg- Rotation of basis vectors, in degrees
[in]eps- Strain value to apply
  • (-1,1)
[in]stype- Direction of strain application
  • biaxial or armchair or zigzag
