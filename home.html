MSL Suite

A simple library for assisting MD/DFT simulations

Home Documentation Install View on GitHub

Description:

This is a library which is class based, with each class focused on preparing, conducting, and analyzing Molecular Dynamics and Density Functional Theory simulations. The entire suite is written in Ansi C and C++99 making using of only standard libraries and some STL for the C++ codes. The library is modular with a separate class for each of the various MD/DFT codes, and one core set of low-level routines for all classes.

Contents:

  • lib/ Source for shared libraries, for DMol, LAMMPS, MSL, and VASP

  • include/ Headers containing definitions of libraries

  • examples/ Directory containing several codes demonstrating the various library methods and there usage

Usage:

See the directory examples/libxxx for some examples of calls to the various library subroutines. This directory also has a Makefile template for use in building your own codes against the library.

Alternatively, you can build your programs with the libraries as follows,

 icpc -L$(LIB_DIR)/libs -llammps -lmathns ... -I$(INC_DIR)/include code.cpp -o out.x
